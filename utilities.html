Command line utilities¶

Release:0.1
Date:June 16, 2013

Wormtable is distributed with two command line utility programs, vcf2wt and wtadmin. These utilities provide a convenient means of creating wormtables from VCF data, and to administer existing wormtables.

vcf2wt¶

Converts data files written in the Variant Call Format (VCF) to wormtable. See vcf2wt –help for detailed program options.

wtadmin¶

Administration tool for wormtable. Provides commands to add and remove indexes, list the indexes present, show the columns in the table and to dump rows from the table stdout. See wtadmin –help for details and program options.
